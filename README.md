# TTS

## Recipe-DNNTTS.ipynb
DNNを基にしたTTSモデルを学習・予測するための一連の実装

### DNN音声合成システム構築の流れ
```
【学習】
フルコンテキストラベル -> 継続長モデル前処理 -> （音素単位）言語特徴量 -> 正規化 -> 継続長モデル学習                                  　
                                        -> （音素単位）音素継続長 -> 正規化 -> ..
音声データ -> 音響モデル前処理 -> （フレーム単位）言語特徴量 -> 正規化 -> 音響モデル学習                          
                            -> （フレーム単位）音響特徴量 -> 正規化 -> ..
```
```
【生成】
テキスト -> 言語特徴量の抽出 -> 継続長の予測 -> 音響特徴量の予測 -> 波形生成 -> 合成音声
```
